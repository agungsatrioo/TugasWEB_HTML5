HTML5 Sebuah evolusi web.

Memperkenalkan Era Baru dalam Web

Bosan dengan teknologi HTML yang itu-itu saja? Kabar baik untuk Anda! Kini versi baru dari HTML, yaitu HTML5, telah hadir. HTML5 hadir dengan beragam fitur baru. Membuat Web Anda lebih atraktif dan memikat. HTML5 menjadikan Web Anda lebih dari sekadar web.

Lebih Interaktif dan Atraktif!

Web yang desainnya indah saja tidak cukup! Buat web Anda lebih interaktif dan atraktif dengan HTML5! Dan web Anda menjadi lebih memikat. Dan tanpa banyak tergantung pada Javascript!

HTML5 + CSS3 = Kesempurnaan

Jangan hanya puas dengan HTML5 saja. Percantik halaman Anda dengan sentuhan seni dari CSS3 yang membuat Web Anda menjadi mahakarya sempurna.

Ingat yang Ada HTML5-nya!

Cukup mudah untuk Anda ketahui jika suatu halaman menggunakan HTML5! Tag-tag baru, fitur-fitur baru, dan tentu saja... <!DOCTYPE html>

Mengapa ?

Lebih optimal

HTML5 dirancang agar berjalan optimal, hampir tanpa bug yang mengganggu kenyamanan Anda.

Lebih efisien

Dengan HMTL5, Anda tidak perlu lagi menuliskan kode terlalu banyak untuk membuat halaman yang menurut Anda menarik. Cukup dengan beberapa baris saja, Anda sudah membuat halaman Web layaknya seperti profesional.

Lebih mudah diakses mesin pencari

HTML5 juga membuat web Anda mudah dicari oleh mesin pencari besar seperti Google, sehingga jutaan pasang mata di belahan dunia bisa melihat Web Anda.

Lebih spesifik

Di HTML5, tidak usah menggunakan <div> untuk membuat bagian konten-konten utama. Ada <header> dan <article> yang siap menunjang produktifitas Anda.

Dukungan multimedia mandiri

Fitur Baru!

Buat apa susah-susah mencari pemutar multimedia pihak ketiga untuk Web Anda jika HTML5 sudah menyediakan itu untuk Anda? Lagipula, ini akan mempersingkat
waktu muat Web Anda dan tentunya, menghemat memori.

Buat permainan sesuai keinginan Anda!

Fitur Baru!

Di HTML5, Anda bisa membuat permainan sesuai keinginan Anda. Bahkan, Anda bisa membagikan kepada semua orang melalui situs Anda!

Penyimpanan yang lebih baik

Teknologi efisiensi HTML5 membuat kuki dan cache dapat disimpan secara baik tanpa memboroskan ruang disk Anda.
